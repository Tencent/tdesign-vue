基础气泡确认框 
使用说明文字及操作按钮对较简单的操作进行二次确认。
不同图标的气泡确认框 
在说明文字之前增加图标，如普通、警示及告警等图标，增强表达以更好的引起用户注意。


带描述的气泡确认框 
在主要说明文字之外增加了操作相关的详细描述，对较复杂的，可能造成疑惑的操作进行详细描述。
自定义气泡确认框按钮 
通过 confirmBtn 和 cancelBtn 属性来自定义确认和取消按钮，支持传入 Button 组件属性或使用 slot 方式。
控制气泡确认框位置 
通过 popupProps 可以透传弹窗底层依赖的 Popup 组件所有已支持的属性，比如控制弹窗出现的位置。

        何时使用 
        需要用户进行二次确认时；
        需要给用户展示反馈信息时。
        与页面布局相关 
        气泡确认框可根据页面需要，进行各方向的自由布局，出现位置应避免遮挡页面信息、或被遮挡。 
        图示：右侧空间不足时，出现在左侧，避免起泡确认框被右侧边缘遮挡
        图示：有相关信息需要展示时，可出现在无关信息的一侧，避免相关信息被遮挡。
        组件搭配使用 
        与全局提示搭配使用，在操作后用来给予用户操作结果反馈。 
        推荐/慎用示例 
        主要说明文字需明确表达其目的及操作的后果。 
        按钮需使用可指引后果的操作词汇，而不是模棱两可的词汇。
